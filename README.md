# LifeRPG

LifeRPG is a fullstack productivity web application that gamifies personal task management using RPG mechanics. Users convert daily tasks into missions that grant experience points and improve character attributes, creating a structured and motivating progression system.

## Concept

The application transforms productivity into an RPG-style experience:

- Create missions (tasks) with rewards
- Gain experience by completing tasks
- Level up as progress increases
- Improve character attributes
- Track history and progress
- Use a modern dark-themed interface

The goal is to motivate consistent improvement through visual feedback and gamification while maintaining a clean and professional user experience.

## Features (MVP)

- User authentication
- Task creation and management
- Daily limits for repeatable tasks
- Experience and level system
- Character attributes
- Progress dashboard
- Task completion history
